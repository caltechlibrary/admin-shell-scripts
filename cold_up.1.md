# NAME

cold_up.bash

# SYNOPSIS

cold_up.bash

# DESCRIPTION

cold_up.bash brings cold up in an orderly fashion. It follows
the following steps

1. Starts the api service (cold_api)
2. Starts the reports system (cold_reports)
3. Starts the front end service (cold)

# OPTION

-h, --help, help
| display this help message

# EXAMPLE

~~~
./cold_up.bash
~~~
