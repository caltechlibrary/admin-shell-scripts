NAME

cold_lookup.bash

SYNOPSIS

cold_lookup.bash C_NAME KEY

DESCRIPTION

cold_lookup.bash will look up the KEY record in the C_NAME (dataset collection) using the COLD API. This provides you with a quick way to see the record that will show up for a person in the COLD web application.
