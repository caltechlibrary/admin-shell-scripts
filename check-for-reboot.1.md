# NAME

check-for-reboot.bash

# SYNOPSIS

check-for-reboot.bash

# DESCRIPTION

This command will check if Ubuntu is expecting to be rebooted.

# EXAMPLES

Here's the typical use case

    sudo apt update;sudo apt upgrade
    sudo apt autoremove; sudo apt autoclean
    check-for-reboot.bash
