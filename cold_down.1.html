NAME

cold_down.bash

SYNOPSIS

cold_down.bash

DESCRIPTION

cold_down.bash brings cold down in an orderly fashion. It follows the following steps

  1. Stops the front end service (cold)
  2. Stops the reports system (cold_reports)
  3. Stops the api service (cold_api)
